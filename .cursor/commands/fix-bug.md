# Исправь ошибку в реализации

## Обзор

Пошаговое исправление найденной ошибки (бага) в реализации

## Шаги
1. **Анализ**
   - Изучи "Шаги для воспроизведения", "Фактическое поведение" (ФР, ФП) и "Ожидаемое поведение" (ОР, ОП) от оператора
   - Спланируй шаги по исправлению. Если нужно — запусти браузер через chrome-devtools и воспроизведи проблему
   - Перепроверь шаги, что они действительно приведут тебя к ожидаемому поведению системы

2. **Реализация**
   - Если тесты отсутствуют, то напиши тесты, которые позволят проверить корректность логики работы. После первого прогона — тесты должны падать, т.к. текущее поведение системы не ожидаемое.
   - Реализуй всё необходимое для исправления ошибки
   - Убедись, что тесты проходят и pnpm validate не возвращает ошибок
   - Проверь итоговый результат в браузере — фактическое поведение системы должно соответствовать ожидаемому

## Чек-лист готовности исправления ошибки
- [ ] Исправление реализовано
- [ ] Исправление проверено через браузер и система работает ожидаемым образом
- [ ] Тесты написаны и проходят
- [ ] Ошибок в коде, линтере и статическом анализе нет
