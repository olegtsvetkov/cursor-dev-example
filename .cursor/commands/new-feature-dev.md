# Реализация новой фичи

## Обзор
Пошаговая реализация новой фичи по PRD

## Шаги
1. **Подготовка**
   - Изучи требования в переданном тебе PRD файле
   - Спланируй шаги по реализации
   - Перепроверь шаги, что они действительно приведут тебя к итоговому состоянию PRD

2. **Реализация**
   - Напиши тесты, которые позволят проверить корректность логики работы
   - Реализуй всё необходимое по PRD
   - Убедись, что тесты проходят и pnpm validate не возвращает ошибок
   - В браузере проверь итоговый результат через критерии приёмки из PRD

## Чек-лист готовности фичи
- [ ] Решение реализовано
- [ ] Решение проверено через браузер и работает требуемым образом
- [ ] Тесты написаны и проходят
- [ ] Ошибок в коде, линтере и статическом анализе нет
- [ ] Критерии приёмки выполнены
